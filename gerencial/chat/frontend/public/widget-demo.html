💬 Chat Widget Demo

Widget de chat estilo WhatsApp para o Sistema de Protesto

🚀 Como Usar

Adicione o script ao seu HTML com os atributos de configuração:


src="/chat-widget.js"
        data-chat-url="http://localhost:5173"
        data-primary-color="#25D366"
        data-width="380"
        data-height="600">



src="/chat-widget.js">
